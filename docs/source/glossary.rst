========
Glossary
========

.. glossary::

    prestans3 attribute
        a prestans3 |Structure| may contain one or more |attributes|\ .

    prestans3 type
        in general this refers to the subclass of a prestans3 :class:`type <prestans3.types>`

    property rule
        a property rule is defined as a function that accepts an |ImmutableType| as its first argument and a
        configuration for the rule as the second.

    scalar
        represents a basic native value. e.g. int, string, float, date, time. immutable by nature and does not contain
        unrelated values.
